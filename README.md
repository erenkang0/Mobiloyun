# BRICLE 🧱

Rastgele renkli Lego tuğlalarıyla yaratıcı yapılar inşa et!

## Özellikler

- **Serbest Yap**: İstediğin yapıyı özgürce inşa et
- **Meydan Oku**: Verilen şekli tuğlalarla doldur, bonus puan kazan
- Sürükle-bırak + dokunarak yerleştirme
- Tuğlaları döndürme
- Geri alma (undo)
- 3D Lego stud görünümü
- Animasyonlu konfeti tebrik ekranı

## Kurulum

```bash
flutter pub get
flutter run
```

## Tech Stack

- Flutter 3.22+
- Riverpod (state management)
- Google Fonts – Fredoka One
